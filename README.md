# Pagina---Catalogo-de-tienda-de-Electronicos
Pagina que funciona como catalogo trayendo de la base de datos los productos electronicos en stock. Los productos se pueden agregar, modificar o eliminar si el usuario esta registrado como root. La arquitectura funciona con React + Express + MySQL comunicadas con una API Rest.

Como principales liberias:
prisma
react-hook-form
swagger
axios
jsonwebtoken
bcrypt
